Chạy chương trình thuật toán:

    - Để chạy giải thuật SOS - ACO -> Trong terminal nhập : python run.py.

    - Chạy thuật toán ACO-LO -> Trong terminal nhập : python ACO_LO.py.

    - Chạy thuật toán ACO -> Trong terminal nhập : python ACO.py.

* Nếu cần thiết , có thể thay đổi các tham số đầu vào trong file  "parameter.py".
